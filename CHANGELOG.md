# Changelog

## [2.0.0] - 2025-12-31

### 추가
- 230개 이상의 KRW 마켓 코인 동시 분석 및 거래 (TARGET_COINS=ALL)
- 실시간 웹 대시보드 (Socket.io 기반)
- 번들 리밸런싱 제안 시스템
- 긴급 뉴스 알림 기능
- 신호 강도에 따른 동적 투자금액 조절
- 부분 매도 기능 및 통계 추적

### 변경
- 리밸런싱 제한 추가 (10분 보유, 5분 쿨다운, -0.5% 손실 제한)
- 수수료 계산 로직 개선 (매수/매도 모두 0.05% 적용)
- API 에러 핸들링 강화

### 수정
- RSI 표시 오류 수정
- 과도한 리밸런싱으로 인한 수수료 손실 문제 해결

- 업비트 API 연동

- 뉴스 감성 분석
- 유전자 알고리즘 파라미터 최적화
- 백테스팅 엔진
- 모의투자 모드
