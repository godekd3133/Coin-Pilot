# 보안 정책

## 취약점 신고

보안 취약점을 발견하셨다면 공개 이슈 대신 비공개로 알려주세요.

연락 방법:
- GitHub Security Advisory 사용
- 또는 저장소 소유자에게 직접 연락

## 주의사항

이 프로젝트는 암호화폐 거래를 다루므로 다음 사항을 지켜주세요:

1. **API 키 보호**: `.env` 파일을 절대 커밋하지 마세요
2. **출금 권한 비활성화**: API 키 발급 시 출금 권한은 제외하세요
3. **모의투자 우선**: 실전 투자 전 반드시 DRY_RUN 모드로 테스트하세요
4. **투자 한도 설정**: 손실 가능한 금액만 사용하세요
